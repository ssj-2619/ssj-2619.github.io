# Garage — FH6 Coverage Tracker

A single-file, offline-first tracker for **Forza Horizon 6**. It keeps track of which cars you own for every Race Type × Class combination, and which Super Wheelspin cars you've collected — so you always know what you're missing.

Everything runs locally in the browser: no backend, no account, no shared database. Each person who opens it gets their own private garage on their own device.
